# Contributing

Thank you for your interest in contributing to Mita.GithubCopilotRtl.

Guidelines
- Fork the repository and create a feature branch.
- Open a pull request with a clear description of your changes.
- Include small, focused commits and add tests where applicable.

Code style
- Follow existing project conventions. This solution targets .NET Framework 4.7.2.

Reporting issues
- Open an issue with steps to reproduce, expected behavior, and screenshots if helpful.
